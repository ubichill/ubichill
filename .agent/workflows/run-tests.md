# テスト・検証ワークフロー

コードの品質を担保するために、テストとLintを実行します。

## 手順

1. Lintの実行
   全パッケージのLintを実行します。
   ```bash
   // turbo
   pnpm lint
   ```

2. ビルド確認
   全パッケージが正常にビルドできるか確認します。
   ```bash
   // turbo
   pnpm build
   ```

3. テストの実行 (将来実装予定)
   現在はプレースホルダーですが、将来的には以下のコマンドで単体テストを実行します。
   ```bash
   pnpm test
   ```
